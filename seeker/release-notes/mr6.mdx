## Version

Global (devices with serial numbers beginning with `SM02G...`): `Seeker.251025.001`

EEA (devices with serial numbers beginning with `SM02E...`): `Seeker.2501027.001`

_You can view your Seeker device serial number in “Settings > About phone”, and then tapping on “Model & hardware”_

## Release notes

🔐 Security

- September 2025 Android security patches

🔑 Seed Vault

- Fix for a rare issue which could cause seeds stored in Seed Vault to be renamed (for example, a seed named “Seeker Seed 1” might be unexpectedly renamed to “Seeker Seed 2”)

  _If you were affected by this issue, please see [this article](https://intercom.help/solana-mobile/en/articles/12651242-how-to-re-import-renamed-seeds-to-restore-seed-vault-wallet-functionality) for guidance on how to restore full Seed Vault Wallet functionality for the renamed seeds_

🛠️ System & Performance

- Fix for a compatibility issue with some Apple-branded chargers
- General stability & bug fixes

📥 Update Size

- Approximately 40 MB (Global) / 42 MB (EEA)
