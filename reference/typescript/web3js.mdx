Interact with accounts and programs on the 
Solana network through the Solana [JSON RPC API](https://docs.solana.com/api/http).

<Button href="https://solana-labs.github.io/solana-web3.js/">API Reference</Button>

### Install dependencies

<CodeGroup>
```shell yarn
yarn add \
  @solana/web3.js \
  react-native-get-random-values \
  react-native-url-polyfill \
  @craftzdog/react-native-buffer
```

```shell npm
npm install \
  @solana/web3.js \
  react-native-get-random-values \
  react-native-url-polyfill \
  @craftzdog/react-native-buffer
```
</CodeGroup>

### Polyfills

Enable all of them in `index.js`

```typescript
import 'react-native-get-random-values';
import 'react-native-url-polyfill/auto';
import {Buffer} from '@craftzdog/react-native-buffer';
global.Buffer = Buffer;
```
