# Solana Mobile Documentation

This repository contains the official documentation for Solana Mobile.

## Development

Install the [Mintlify CLI](https://www.npmjs.com/package/mint) to preview the documentation locally:

```bash
npm i -g mint
```

Run the following command at the root of the documentation directory:

```bash
mint dev
```

## Publishing Changes

Changes pushed to the main branch will be automatically deployed to production.
